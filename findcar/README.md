## 北洋找车

#### 背景
现在越来越多的人开始拼车回自己的老家，方便、快捷。但是目前大部分都是通过微信群进行找车或者车找人。群里有时发布一些无关的信息，导致消息不能准确的让我门浏览到。此小程序简单的实现了发布人找车和车找人的功能，在首页可以看到用户们发布的信息，然后可以在用户页面对自己发布的信息进行删除。
#### 主要功能
- 广场
	- 用于展示发布信息
	- 通过目的地搜索信息
	- 通过车找人和人找车筛选信息
	- 点击信息进入详情也，点击马上联系即可自动跳转到拨号页面
	- 备注
- 发布
	- 发布信息
	
- 我的
	- 发布历史
	- 个人资料
	- 信息反馈
	- 关于
